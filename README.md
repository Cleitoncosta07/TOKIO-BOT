## 🌟 Funcionalidades do TOKIO BOT

### 🔒 Moderação Ultra God
- Anti-call (bloqueia chamadas indesejadas)
- Anti-imagem em visualização normal
- Anti-mencionar grupo
- Anti-bots
- Anti-spam e links maliciosos
- Gerenciamento de grupos completo (kick, ban, reset, avisos)

### 🎭 Entretenimento
- Música (YouTube, Spotify)
- Memes, imagens e gifs
- Pesquisa de notícias e curiosidades
- Stickers e efeitos visuais

### 🤖 Inteligência Artificial
- Chatbot com respostas inteligentes
- Comandos de diversão com IA
- Processamento de linguagem natural
- Personalização de respostas por usuário

### 💾 Downloads e Integrações
- Twitter, Mediafire, Google Drive
- Extração de áudio e vídeo
- Conversão de arquivos para formatos compatíveis

### ⚙️ Configurações
- Prefixos customizáveis
- Mensagens de boas-vindas
- Configuração de API VEX e tokens
- Controle total do bot via comandos internos

### 🛡️ Segurança e Backup
- Backup automático de dados e chats
- Logs de eventos e moderação
- Proteção contra crashes e erros comuns

---

📱 Como conectar

1. Escolha Conectar via QR Code ou Conectar via Código no menu do bot.


2. Escaneie o QR Code com o WhatsApp ou insira o código gerado.


3. O bot estará pronto para rodar e moderar automaticamente.




---

👥 Moderação Ultra God

Recurso	Descrição

Anticall	Bloqueia ligações para o bot
Anti imagem	Bloqueia imagens fora do padrão permitido
Antimencionar	Bloqueia menções de todos no grupo
Antibots	Detecta e remove bots automaticamente
