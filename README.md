## 🛠️ Instalação

```bash
# Clone o repositório
git clone https://github.com/seuusuario/TOKIO-BOT.git
cd TOKIO-BOT

# Instale as dependências
npm install

# Inicie o bot
sh start.sh

> ⚠️ Importante: Certifique-se de ter o Node.js LTS instalado, FFmpeg e Git.




---

📱 Como conectar

1. Escolha Conectar via QR Code ou Conectar via Código no menu do bot.


2. Escaneie o QR Code com o WhatsApp ou insira o código gerado.


3. O bot estará pronto para rodar e moderar automaticamente.




---

👥 Moderação Ultra God

Recurso	Descrição

Anticall	Bloqueia ligações para o bot
Anti imagem	Bloqueia imagens fora do padrão permitido
Antimencionar	Bloqueia menções de todos no grupo
Antibots	Detecta e remove bots automaticamente
