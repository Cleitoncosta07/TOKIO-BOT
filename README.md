## 🚀 Primeiros Passos: Guia Rápido

<details>
<summary><strong>⚡ Início rápido para quem tem experiência</strong></summary>

1. Prepare o ambiente: instale **Node.js (versão 18 ou superior)** e **Git**.  
2. Clone o repositório oficial do TOKIO Bot.  
3. Use um **número de telefone secundário** para o bot.  
4. Após a conexão, digite `/menu` no WhatsApp para ver os comandos.
</details>

## 🔌 Conectando o Bot ao WhatsApp

<details>
<summary><strong>📱 Métodos de conexão</strong></summary>

1. **QR code**:  
   - Execute `npm start`.  
   - No WhatsApp, vá em **Configurações → Aparelhos Conectados → Conectar um Aparelho** e escaneie o QR code.  

2. **Código de pareamento**:  
   - Execute `npm start`.  
   - No WhatsApp, vá em **Configurações → Aparelhos Conectados → Conectar com Número de Telefone** e insira o código.
</details>

## ❓ Resolvendo Problemas Comuns (FAQ)

<details>
<summary><strong>📋 Dúvidas frequentes e soluções</strong></summary>

- **`npm: command not found`**: Verifique se Node.js e npm estão instalados (`node -v`, `npm -v`).  
- **QR code expirou**: Execute `npm start` novamente.  
- **Problemas de conexão**: Verifique internet, número de telefone e permissões no diretório.
</details>
