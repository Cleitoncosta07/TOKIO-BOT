## 🚀 Instalação Termux
<img src="https://user-images.githubusercontent.com/108157095/182052725-6568419a-6a9f-490a-85ea-90b94af694fe.png" height="25px">

**1° Comando**
```bash
pkg update -y && pkg upgrade -y && pkg install git nodejs ffmpeg wget tesseract -y
```

**2° Comando**
```bash
termux-setup-storage
```

**3° Comando**
```bash
cd /storage/emulated/0/Download && git clone https://github.com/Cleitoncosta07/TOKIO-BOT
```

**4° Comando**
```bash
cd /storage/emulated/0/Download/TOKIO-BOT && npm i && npm start
```
