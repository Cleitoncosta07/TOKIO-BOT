## 📥 Instalação Completa

### 📱 Termux

apt update -y && apt upgrade -y  
pkg install git -y  
pkg install nodejs -y  
pkg install ffmpeg -y  
pkg install wget -y  
pkg install tesseract -y  

termux-setup-storage  

cd /storage/emulated/0/Download  
git clone https://github.com/Cleitoncosta07/TOKIO-BOT  

cd TOKIO-BOT  
npm install  
npm start
